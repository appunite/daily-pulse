### Installation and running
1. Create and ctivate the environment
```
python3 -m venv .venv
. .venv/bin/activate
```

2. Install required packages
```
pip install -r requirements.txt
```

3. Set required environment variables. Default values are defined in `.env.default`

4. Run the app
```
flask run
```
